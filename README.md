## Web application for the sale and purchase of cars.

### About the application
+ The site contains ads with a description, brand of car, body type, photo. 
+ The ad has a status - sold or not. 

### Technology stack:
```text
+ Java 17;
+ Maven 4.0;
+ Spring boot;
+ Thymeleaf;
+ Bootstrap 5;
+ PostgreSQL 14;
+ JDBC;
+ Liquibase;
+ H2 1.4;
+ Log4j;
+ JUnit 5;
+ Mockito 4. 
```
