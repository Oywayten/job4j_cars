## Веб-портал по продаже и покупке автомобилей. 

### О проекте:
+ На сайте размещены объявления с описанием, маркой машины, типом кузова, фотографией. 
+ У объявления есть статус - продано или нет. 

### Использованные технологии
```text
+ Java 17;
+ Maven 4.0;
+ Spring boot;
+ Thymeleaf;
+ Bootstrap 5;
+ PostgreSQL 14;
+ JDBC;
+ Liquibase;
+ H2 1.4;
+ Log4j;
+ JUnit 5;
+ Mockito 4. 
```
